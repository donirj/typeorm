TypeORM es un ORM (object relational mapping) para NodeJS, capaz de funcionar
 en muchos ambientes, como el propio Node, Cordova,
  Ionic, Electron y por supuesto NestJS. Es una herramienta 
  que permite trabajar con diversas bases de datos, ayudando
   no sólo gracias a su capa de abstracción, sino también a la
    hora de montar las consultas y realizar operaciones con los
     datos.

ORM es un modelo de programacion que permite mapear las estructuas
 de una base de datos relacional (sql, server, oracle, mysql)
 en adelante, sobre una estructura logica de entidades con el objeto
 de simplificar y acelerar el desarrollo de nuestras apps
